# Multi-Robot-Multi-Queue-Control-via-Exhaustive-Assignment-Actor-Critic-Learning

This repository implements a structure-aware actor-critic framework for real-time scheduling in multi-robot multi-queue systems. The setting considers heterogeneous Bernoulli task arrivals across queues, deterministic unit service times, one-slot travel delays for switching, and one-robot-per-queue feasibility constraints. Building on the exhaustive-service structural property, the policy architecture reduces the action space by fixing the decisions of robots at nonempty queues and learning only the reassignment of idle robots through a masked sequential assignment mechanism. The code includes the simulation environment, PPO training pipeline, policy/value network implementation, and benchmarking utilities used to evaluate discounted holding cost and mean queue length against greedy ESL baselines and small-instance optimal references.
